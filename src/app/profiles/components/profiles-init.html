Profile

Profile definieren, in welchen Formaten und welcher Qualität eine Sendung archiviert wird. Die Qualität einer Aufnahme kann nach einer bestimmten Anzahl Monaten nach der Ausstrahlung in mehreren Schritten reduziert werden und/oder die Aufnahme kann komplett aus dem Archiv gelöscht werden.

Über die maximale öffentliche Bitrate wird bestimmt, welche Wiedergabeformate für nicht eingeloggte Hörer verfügbar sind. Für diese sind nur Wiedergabeformate mit gleicher oder tieferer Bitrate verfügbar.

Das Standardprofil wird automatisch neuen Sendungen zugewiesen.

Bitte wähle links ein Profil zum Bearbeiten.
